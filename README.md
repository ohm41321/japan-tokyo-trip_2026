# 🗼 Tokyo Trip 2026 - Itinerary App

A beautiful, responsive web app displaying your 7-day Tokyo trip itinerary.

## 🚀 Quick Start

### Install Dependencies
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub Integration
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repo
5. Click "Deploy"

### Option 3: Vercel Dashboard
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Upload this folder
4. Click "Deploy"
